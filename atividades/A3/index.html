Tag de texto pré formatado:

A tag de inserção de um texto pré-formatado é "pre".

Por exemplo:

  • Sem pre:

    • Amor é fogo que arde sem se ver; É ferida que dói, e não se sente; É um contamento descontente; É dor que desatina sem doer.

  • Com pre:

    •             Amor é fogo que arde sem se ver;
            É ferida que dói, e não se sente;
            É um contamento descontente;
            É dor que desatina sem doer.

    Tag de links

    A tag serve para inserir links no documento HTML.

    Os atributos são: href (Link), target (Onde o link será aberto).
    O atributo target possui 4 opções:

    • _blank: Abre o link em uma nova janela ou aba
      • Exemplo: Google link

    • _self: Abre o link na mesma janela (Padrão)
      • Exemplo: Google link

    • _parent: Abre o link no quadro pai
      • Exemplo: Google link

    • _top: Abre o link no corpo completo da janela
      • Exemplo: Google link

    Tag table (Tabelas)

    Uma tabela é definida por outras tags:

    • tr: Define uma linha
    • td: Define uma célula de dados
    • th: Define a célula de um cabeçalho
    • thead: Agrupa o conteúdo do cabeçalho
    • tbody: Agrupa o conteúdo do corpo
    • tfoot: Agrupa o conteúdo do rodapé

    Exemplo:

    Produto Valor
    Gabinete 100 R$
    Processador intel i7 500 R$
    Motherboard 230 R$
    Valor total 830 R$

    Expansão de coluna da tabela

    Com o atributo colspan da tag th é possível redimensionar o tamanho de uma coluna/célula

    Exemplo:

    Nome Telefone
    Kevin (11)5555-5555 (11)7777-7777

    Expansão de linha da tabela

    Com o atributo rowspan da tag th é possível redimensionar o tamanho de uma linha

    Exemplo:

    Nome Kevin
    Telefone (11)5555-5555
    (11)7777-7777

    Citação e abreviação

    Definição de citação e abreviação

    tags:

    • q: Citação breve

      • Exemplo:

      Segundo Friedrich Nietzsche: A vida vai ficando cada vez mais dura perto do topo.

    • blockquote: Citação de outra fonte

      • Exemplo:

      Só há duas maneiras de viver a vida: a primeira é vivê-la como se os milagres não existissem. A segunda é vivê-la como se tudo fosse milagre.

    • abbr: Abreviatura ou acrônimo

      • Exemplo:

      O Enem foi instituído em 1998, com o objetivo de avaliar o desempenho escolar dos estudantes

    • address: Elemento de contato

      • Exemplo:

      Endereço: Rua Pedro Vicente, nº 625 - Canindé, São Paulo - SP. Site: spo.ifsp.edu.br

    • cite: Título de obra

      • Exemplo:

      Monalisa por Leonardo Da Vinci, também conhecida como A Gioconda.

    • bdo: Direção do texto

      • Esse possui dois atributos: rtl (Da direita para a esquerda), e ltr (Da esquerda para a direita)

      Exemplo:

      • rtl
        • Texto da direita para esquerda
      • ltr
        • Texto da esquerda para direita

    Video

    Para inserir um video em um documento HTML, utiliza-se a tag "video" que possui os seguintes atributos:

    • width: Determina o comprimento
    • height: Determina a altura
    • controls: Exibe as opções de controle do vídeo
    • autoplay: Inicia o vídeo ao carregar a página
    • src: Define a URL do vídeo (Dentro da tag source)

    Favicon

    Favicon é a junção da palavra favorite e icon. Com essa tag é possível a inserção de icones ou seja, a insercão de pequenas imagens. A tag link possui os seguintes atributos:

    • rel: Indica o tipo de arquivo que foi inserido
    • href: Determina o local da imagem
    • size: Determina o tamanho da imagem
    Essa tag deve ficar no head do documento HTML.
